# 一、最核心、必做的 4 种题型
单选题（SingleChoice）
4 个选项，只有 1 个正确
最常用、最好做
多选题（MultipleChoice）
多个正确答案
前端需要做复选框
判断题（TrueFalse）
只有 正确 / 错误
简单、适合基础题
填空题（FillBlank）
填写答案文本
可做严格匹配或模糊匹配
# 二、进阶题型（可选加，让项目更完整）
简答题（ShortAnswer）
纯文字回答
前端用多行输入框，后端只存答案不判分（或人工判分）
编程题（Coding）
Java / Python / 算法题
输入代码片段，后端可对接判题引擎（简单版可只存参考答案）
情景分析题（Scenario）
给一段业务场景，回答问题
适合面试真实场景
